# 🎧 Spotify Data Engineering & Analytics Project

## 📌 Overview

This project demonstrates an end-to-end data pipeline using MongoDB, Python, and Power BI to analyze Spotify music data.

## ⚙️ Tech Stack

* MongoDB (Database)
* Python (Data Processing)
* Power BI (Visualization)
* Excel (Data Cleaning)

## 🔄 Data Pipeline

CSV → MongoDB → Python (ETL) → CSV → Power BI Dashboard

## 🚀 Features

* Data storage using MongoDB
* Data extraction and transformation using Python (pandas)
* Interactive Power BI dashboard
* Insights on song popularity, artists, and trends

## 📊 Dashboard Preview

![Dashboard](dashboard.png)

## 🧠 Key Insights

* Medium popularity songs dominate the dataset
* High popularity songs are fewer but impactful
* Explicit content shows a steady trend

## 📂 Files Included

* `fetch_data.py` → Python ETL script
* `cleaned_spotify_data.csv` → Processed dataset
* `Spotify_Analytics_Dashboard.pbix` → Power BI dashboard

## 📌 Conclusion

This project showcases both Data Engineering and Data Analytics concepts using real-world data.
